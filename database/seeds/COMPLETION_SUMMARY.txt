## 🚀 How to Use

### Quick Start (3 commands)

```bash
# 1. Ensure database is ready
rails db:migrate

# 2. Run all seeds
for seed in database/seeds/{00,001,002,003}_*.sql; do 
  psql -U postgres -d tenadam_development -f "$seed"
done

# 3. Verify
psql -U postgres -d tenadam_development -c "SELECT COUNT(*) FROM patients;"
```
