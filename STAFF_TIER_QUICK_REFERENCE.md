## Minimum Staffing by Tier

### Health Post (11 staff minimum)
- **1x** General Practitioner
- **1x** Registered Nurse
- **1x** Practical Nurse
- **1x** Pharmacist
- **1x** Lab Technician
- **2x** Nursing Aides/Support
- **1x** Hospital Administrator
- **1x** Training Coordinator
- **1x** Security Guard
- **1x** Housekeeping Staff
- **1x** Community Health Worker

### Health Center (36 staff minimum)
- **2x** General Practitioners
- **1x** Medical Director
- **3x** Registered Nurses
- **1x** Lab Director
- **1x** Sonographer
- **1x** Midwife
- Plus: Pharmacist, Dietitian, Social Worker, Admin staff, etc.

### Primary Hospital (103 staff minimum)
- **2x** Attending Physicians
- **1x** General Surgeon
- **1x** Anesthesiologist
- **1x** Intensivist (ICU)
- **8x** Registered Nurses (various specialties)
- **1x** Lab Scientist
- **1x** Radiographer
- Plus: Full administrative, IT, allied health teams

### General Specialized Hospital (75 staff minimum)
- **3x** Specialists (Cardiology, Psychiatry, Oncology, etc.)
- **2x** Advanced Surgeons (Orthopedic, Neuro, Plastic, etc.)
- **1x** Pathologist
- **1x** Research Scientist
- Plus: All roles from Primary Hospital

### National Health System (82 staff minimum)
- **CEO + C-Suite** (CMO, CNO, CFO, CIO)
- **1x** Cardiothoracic Surgeon
- **All Specialties** represented
- **Research Infrastructure**
- Plus: All roles from General Specialized

## Super Admin Approval Checklist

When an organization requests staff role, Super Admin should verify:

```
☐ Organization Tier Qualification
  └─ Does tier allow this role?
  
☐ Credibility Assessment
  └─ Does org actually provide this service?
  └─ Has org been flagged for misconduct?
  
☐ Minimum Staffing Met
  └─ Does org have prerequisite staff?
  └─ Example: Can't have Cardiologist without Hospital infrastructure
  
☐ Service Offerings Match
  └─ Did org apply for this service?
  └─ Example: Health Center claiming no ER shouldn't have ER staff
  
☐ Tier Progression
  └─ Is org attempting to jump tiers?
  └─ Example: Health Post requesting Cardiothoracic Surgeon
```

## Database Queries for Admin

### Find all pending staff requests
```sql
SELECT 
  org.name,
  st.title,
  ostr.justification,
  ostr.created_at
FROM organization_staff_template_requests ostr
JOIN organizations org ON ostr.organization_id = org.id
JOIN staff_role_templates st ON ostr.staff_template_key = st.template_key
WHERE ostr.status = 'pending'
ORDER BY ostr.created_at DESC;
```

### Check what staff org currently has approved
```sql
SELECT 
  st.title,
  st.category,
  ostr.status,
  ostr.approved_at
FROM organization_staff_template_requests ostr
JOIN staff_role_templates st ON ostr.staff_template_key = st.template_key
WHERE ostr.organization_id = $1
AND ostr.status = 'approved'
ORDER BY st.sort_order;
```

### Verify minimum staffing for tier
```sql
SELECT 
  st.title,
  sta.min_staff_required,
  COUNT(DISTINCT u.id) as current_count,
  CASE 
    WHEN COUNT(DISTINCT u.id) < sta.min_staff_required THEN '⚠️ UNDER'
    ELSE '✅ MET'
  END as status
FROM staff_template_tier_access sta
JOIN staff_role_templates st ON sta.staff_template_key = st.template_key
LEFT JOIN org_staff_profiles osp ON st.template_key = osp.staff_template_key
LEFT JOIN users u ON osp.user_id = u.id
WHERE sta.organization_tier = 'health-center'
GROUP BY st.title, sta.min_staff_required;
```
